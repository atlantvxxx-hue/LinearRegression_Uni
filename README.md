# LinearRegression_Uni

Анализ данных для предсказания вероятности поступления в университет на основе оценок, исследований и других факторов. Проведена загрузка данных, предварительный анализ, визуализация, подготовка  и построение модели линейной регрессии. Использовались библиотеки: pandas, matplotlib, seaborn, scikit-learn (sklearn). 

Data analysis for predicting the probability of university admission based on grades, research, and other factors. Data loading, preliminary analysis, visualization, preparation, and linear regression model construction were performed. The following libraries were used: pandas, matplotlib, seaborn, and scikit-learn (sklearn).
